Features

パソコン教室ピアの特徴

初心者でも安心の個別指導

一人ひとりの目的とレベルに合わせて内容をカスタマイズ。何度同じ質問をしても大丈夫な、アットホームな教室です。

Excel・Word・PowerPointを基礎から

文字入力などの基本から、実務で使える操作まで。よく使う機能を中心に、無理なく身につけられます。

MOS資格対策にも対応

Excel・Word・PowerPointのMOS資格取得をサポート。試験の傾向をお伝えしながら、基礎から対策します。

仕事・再就職・地域活動の資料作成まで

事務職への再就職、仕事で使う資料づくり、PTA・町内会・自治会の資料作成まで、目的に合わせて学べます。

通いやすい月謝制

わかりやすい月謝制で、追加費用の心配が少ない料金設計。続けやすさにこだわっています。

土日祝日・夜間にも対応

お仕事帰りや週末にも通いやすい時間帯に開講。ご都合に合わせて受講できます（教室により異なります）。

First time

はじめてでも大丈夫です

パソコン教室ピアには、「パソコンが苦手」「何度も同じことを聞いてしまいそう」「仕事で使えるか不安」という方も多く通われています。電源の入れ方や文字入力から、Excel・Wordの実務的な使い方、資格取得まで、目的に合わせて少しずつ学べます。

パソコン教室ピアは、国分寺教室・八王子教室の2教室で、初心者の方にも分かりやすい個別指導を行っています。国分寺周辺でお探しの方はパソコン教室ピア 国分寺教室、八王子周辺でお探しの方はパソコン教室ピア 八王子教室をご覧ください。

国分寺教室はJR国分寺駅北口から徒歩2分、八王子教室はJR八王子駅から徒歩4分です。どちらの教室でも無料体験レッスンを実施していますので、お近くの教室へお気軽にご相談ください。

まずは教室の雰囲気を見に来ませんか？

パソコン教室ピアでは、国分寺教室・八王子教室のどちらでも無料体験レッスンを実施しています。現在のお悩みや学びたい内容をお伺いし、どのように学べるかをご案内します。無理な勧誘はありませんので、お近くの教室へお気軽にご相談ください。
